# Changelog

All notable changes to this project will be documented in this file.

## [1.1.0] - 2026-02-27

### Changed
- Replaced ALL emoji characters with SVG icons (values, testimonials, contact, mobile close)
- Redesigned value-icon containers to properly house SVG icons

### Added
- **Dynamic Animated Scrollbar**: Custom gradient thumb with continuous animation + scrollbar glow that follows thumb position with color-shifting hue
- **Random Floating Shapes**: Geometric shapes (circles, rings, diamonds, crosses, dots) that continuously float upward with rotation
- **Hero Animated Rings**: 3 concentric pulsing rings behind hero content
- **Hero Typing Effect**: Badge text cycles through phrases (n8n Experts, AI-Powered Solutions, etc.)
- **Card Shine Sweep**: Light sweep effect on service card hover
- **CTA Pulse Glow**: Subtle blue glow pulse on Get Started button
- **Project Card Accent**: Left border animates on hover
- **Wave Section Dividers**: SVG wave shapes separating sections
- **Grid Slide Animation**: Hero perspective grid slowly moves
- **Shimmer Text**: Gradient highlight on hero text animates
- **Section Label Pulse**: Line width animates on section labels
- **Stats Bar Glow**: Radial glow animation in stats section
- **Staggered Card Entrances**: Cards enter with cascading delay
- **Random Stat Micro-Pulse**: Random stat item glows every 4 seconds
- **Section Parallax**: Subtle Y-offset on section headers while scrolling
- **BG Dots Pattern**: Dot grid in How It Works section
