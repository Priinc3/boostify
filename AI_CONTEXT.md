# AI Context for Boostify Claude (Redesigned Website)

## Project Overview
- **Purpose**: Modern redesign of Boostify Corp website — AI & Automation Agency specializing in n8n workflow automation
- **Tech Stack**: Vanilla HTML, CSS, JavaScript (no frameworks, no build tools)
- **Architecture**: Single-page static website with smooth-scroll navigation

## Current State
- **Version**: 1.0.0
- **Status**: In Development
- **Last Updated**: 2026-02-27

## File Structure
```
boostify_claude/

├── style.css       # Complete design system (navy/white theme)
```
